# Nội dung bài tập

Bài tập gồm các class sau

- Customer
- CustomerType
- MonthlyUsage
- MonthlyStatement

Mục đích của bài tập nhằm tính ra tiền phí điện thoại hàng tháng của khách hàng căn cứ trên:

- Thời lượng gọi
- Số lượng tin nhắn

# Yêu cầu

- Refactore lại code nhằm loại bỏ Switch bằng Polymorphism
